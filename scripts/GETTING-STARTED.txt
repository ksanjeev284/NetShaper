NetShaper — Getting started
===========================

EASIEST — GUI installer
-----------------------
1. Download NetShaper-Setup-*.exe from GitHub Releases
2. Double-click → accept UAC → Next through the wizard
3. Optionally enable "Add CLI to PATH" and desktop shortcut
4. Finish and launch NetShaper

PORTABLE ZIP
------------
1. Unzip NetShaper-*-win-x64.zip
2. Right-click Setup.cmd → Run as administrator
   (or double-click Setup.cmd — it will ask for elevation)
3. Choose [1] Install app
4. Start NetShaper from the Desktop or Start Menu
5. Accept the UAC prompt when the app starts

YOUR FIRST LIMIT (in the app)
-----------------------------
1. Open Live traffic — wait 1–2 seconds for rates to warm up
2. Right-click an app → Limit… (or Rules / Limits tab)
3. Enter kbps (e.g. 1000 = ~1 Mbps)
4. Click Apply all (needs Administrator)
5. Shaper mode Soft is the default (QoS + soft pulse)

Without admin you can still store rules and watch traffic; system
enforcement (WFP block/allow, QoS) needs elevation.

OPTIONAL
--------
• Setup option [3] or [4]: install WinDivert for Packet-mode smooth limiting
• CLI on PATH (option [2]):  NetShaper.Cli.exe sample 2
• CLI examples:
    NetShaper.Cli.exe limit chrome 500
    NetShaper.Cli.exe priority chrome high
    NetShaper.Cli.exe lockdown on
    NetShaper.Cli.exe apply-all --persist
    NetShaper.Cli.exe profile list

UNINSTALL
---------
• Apps & Features → NetShaper → Uninstall
• Or Setup.cmd → [5] Uninstall
• Policy/stats under %ProgramData%\NetShaper are kept unless you pass -RemoveData
